## 전체 화면 이동 경로

| 출발 화면 | 액션 | 도착 화면 & 조건 |
|---|---|---|
| 앱 진입 | 자동 | 비로그인 → Onboarding / 로그인 완료 → 홈 (JWT 유효성 확인) |
| Onboarding | 완료 | 홈 (레벨 배정 완료, users DB 저장) |
| 홈 | 복습 알림 탭 | 복습 세션 탭 (review_queue today 조회) |
| 홈 | 빠른 시작 | 마지막 학습 표현의 쉐도잉 플레이어 |
| 카테고리 | 상황 탭 | 상황별 표현 목록 (필터 적용) |
| 표현 목록 | 표현 카드 탭 | 쉐도잉 플레이어 (/shadowing/[id]) |
| 쉐도잉 플레이어 | 녹음 완료 | 발음 평가 요청 → 피드백 패널 (바텀 시트) |
| 피드백 패널 | 재도전 | 동일 쉐도잉 플레이어 (attempt_number+1) |
| 피드백 패널 | 다음 표현 | 다음 expressions.id 쉐도잉 플레이어 |
| 피드백 패널 | 완료 | 표현 목록 복귀 + 진도 업데이트 |
| 진도 탭 | 주간 리포트 | 리포트 화면 (/report/weekly) |
| 설정 | 로그아웃 | Onboarding (JWT 삭제) |
| 대화 시뮬 | 종료 | 홈 복귀 + 시뮬 결과 저장 |

## 핵심 플로우 상세 — 쉐도잉 전체 사이클

| 단계 | 화면 상태 | 로딩 상태 | 에러 상태 |
|---|---|---|---|
| ① 표현 로드 | 표현 텍스트 스켈레톤 표시. TTS URL 프리패치 시작 | 스켈레톤 표시 | 404: '표현을 찾을 수 없습니다' + 목록 복귀 버튼 |
| ② TTS 재생 | 버퍼링 스피너. 청크별 진행 바 | 버퍼링 스피너 | 503: '음성 로드 실패' + 재시도 버튼 |
| ③ 청크 쉐도잉 | 현재 청크 하이라이트 + 일시정지 아이콘 | 없음 (로컬 재생) | — |
| ④ 녹음 시작 | RecordButton 빨간 펄스 애니메이션. 경과 시간 표시 | — | 마이크 권한 없음: 권한 요청 다이얼로그 |
| ⑤ AI 분석 중 | 전체 화면 오버레이 스피너 + '분석 중...' 텍스트 | 오버레이 스피너 | 429: '월 사용량 초과' + 수동 입력 안내 |
| ⑥ 피드백 표시 | ScoreRadar 애니메이션 + FeedbackCard 슬라이드 인 | — | GPT 피드백 없음 시 기본 메시지 |

## 각 화면의 3가지 상태 정의

| 화면 | 로딩 상태 | 에러 상태 | 빈 상태 (Empty State) |
|---|---|---|---|
| 홈 대시보드 | 스켈레톤 카드 3개 | 오프라인 배너 + 캐시 데이터 표시 | 학습 첫날: '첫 표현을 시작하세요' CTA |
| 카테고리 목록 | 스켈레톤 탭+카드 | 재시도 버튼 + 오프라인 안내 | 콘텐츠 준비 중 (미출시 카테고리) |
| 복습 세션 | 스켈레톤 카드 | 재시도 버튼 | 오늘 복습 없음: '모두 완료! 새 표현 학습하기' CTA |
| 진도 통계 | 차트 스켈레톤 | 데이터 없음 안내 | 학습 기록 없음: '첫 학습을 시작하면 통계가 생깁니다' |
| 주간 리포트 | 전체 로딩 스피너 | 생성 중 안내 | 이번 주 학습 없음: '이번 주 학습 후 리포트 생성' |

## 권한별 접근 제어

| 화면 | 비로그인 접근 | 로그인 접근 |
|---|---|---|
| Onboarding | 허용 (앱 첫 진입) | 허용 (레벨 변경 목적) |
| 홈/카테고리/복습/진도 | 차단 → Onboarding 리다이렉트 | 허용 |
| 쉐도잉 플레이어 | 차단 → Onboarding 리다이렉트 | 허용 (본인 표현만) |
| 대화 시뮬 | 차단 | 허용 |
| 커스텀 표현 추가 | 차단 | 허용 (GPT 변환 비용 발생) |
| 주간 리포트 | 차단 | 허용 (본인 데이터만) |
| 설정 | 차단 | 허용 |
