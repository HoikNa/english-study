# **1. 문제 정의 (My Problem Statement)**

## **1.1 나의 현재 상황**

토익 850점으로 독해·어휘 능력은 충분하지만, 실제 말하기 상황에서 머릿속 생각을 영어로 즉각 표현하지 못해 대화가 끊긴다. 고급 표현보다 기초적인 브로큰 잉글리쉬로 넘어가는 상황이 반복되고, 비즈니스 미팅이나 현지 일상에서 자신감이 떨어진다. 단순 어휘·문법 지식과 '유창한 말하기' 사이의 간격을 메우는 것이 핵심 과제다.

## **1.2 구체적 페인포인트**

| **상황** | **문제 현상** | **원하는 결과** |
| --- | --- | --- |
| **IT 비즈니스 미팅** | **의견은 있는데 영어 표현이 안 떠올라 침묵. 결국 단순한 Yes/No로 마무리** | **아이디어를 영어로 즉각 표현, 미팅 주도 가능** |
| **현지 일상생활** | **집 계약·병원·관공서에서 긴장해서 브로큰 잉글리쉬. 상대방 재확인 요청 빈번** | **짧고 명확한 문장으로 원하는 바를 정확히 전달** |
| **IT 기술 설명** | **한국어로는 유창하게 설명 가능한 내용을 영어로 설명할 때 어휘가 막힘** | **IT/통신 기술 개념을 영어로 자연스럽게 설명** |
| **고급 표현 사용** | **알고 있는 표현만 반복 사용. 원어민처럼 자연스러운 표현이 안 나옴** | **상황에 맞는 고급 표현을 쓰는 습관 형성** |
| **발음·억양** | **빠른 원어민 발화를 듣고 이해는 하나, 나의 발음·억양이 어색해서 상대가 되묻는 경우 있음** | **명확한 발음과 자연스러운 억양으로 첫 번에 이해시킴** |

## **1.3 기존 학습 앱의 한계**

- **시험 대비 앱 (토익/토플): 읽기·쓰기 중심. 말하기 훈련 없음**

- **일반 회화 앱: IT 비즈니스·통신 기술 특화 표현 부재**

- **AI 튜터 앱: 피드백이 텍스트 기반. 실제 발음·억양 교정 약함**

- **쉐도잉 앱: 나의 직업·이민 목적에 맞는 맞춤 콘텐츠 없음**

## **1.4 SpeakReadyMY가 해결하는 것**

내 구체적 상황(45세, 통신사 IT 기획자, 가족 이민 후 IT 비즈니스 준비)에 딱 맞춘 표현만 학습한다. OpenAI API를 통해 내 발화를 실시간 분석하고, 내가 실제로 틀리는 패턴에 집중 피드백을 제공하는 완전 개인화 앱이다.

목표 문장: {target_sentence}
Azure 인식 텍스트: {azure_recognized_text}
Azure 점수: 발음 {pron}/100, 유창성 {fluency}/100, 억양 {prosody}/100
다음을 한국어로 응답 (총 200자 이내):
1. 가장 어색한 표현 1가지 지적 (구체적으로)
2. 원어민이 실제로 쓰는 대안 표현 3가지 (예문 포함)

## **4.1 나의 특수 원칙 (개인화)**

- **토익 어휘 활용: 이미 아는 단어를 '말하기'로 전환하는 훈련에 집중 (새 단어 암기 최소화)**

- **브로큰 잉글리쉬 → 고급 표현 트래킹: 내가 자주 쓰는 단순 표현 DB화 → 더 나은 표현으로 점진 교체**

- **IT 기획자 언어 우선: '요구사항 정의, 스프린트, 마일스톤, 이해관계자' 등 내 일상 업무 단어로 시작**

- **20~30분 완성 세션: 하루 학습량을 출퇴근·점심 시간에 완료 가능한 단위로 설계**

- **가족 이민 일상 선행 학습: 초기 3개월은 현지 정착 표현(학교 상담, 병원, 마트) 집중 후 비즈니스로 확장**
